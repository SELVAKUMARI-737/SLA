What is an HTML Table?

HTML table is used to display data in rows and columns (like marks list, timetable, bill, etc.)

example:

Name Mark
Ram 85

border Used to add a border around the table.

cellpadding Adds space between the cell content and the cell border.

cellspacing Adds space between table cells.

rowspan Allows a table cell to span across multiple rows.

colspan Allows a table cell to span across multiple columns.

CSS Properties for Tables

border Sets the border style for table or table cells.

padding Adds inner spacing inside table cells.

text-align Aligns text inside the table cells.

width Controls the width of the table or columns.

border-collapse Decides whether table borders are separated or merged.
