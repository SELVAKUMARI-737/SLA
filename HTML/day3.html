Semantic Tags:

  • Important for Search Engine Optimization
  • Helps screen readers for accessibility
  • Makes code readable and organized

Semantic vs Non-Semantic Tags:

  • Semantic tags are like <header>, <nav>, <main>, <section>, <article>, <aside>, <footer>

  • Non-semantic tags are <div> or <span> — they don’t really say what the content is.
